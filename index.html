🔗 Google 연동 설정

Calendar와 Sheets를 연동하면 일정이 자동 표시되고 데이터가 Sheets에 저장됩니다.

STEP 1 — GOOGLE CALENDAR
1
OAuth Access Token 입력
Google OAuth로 발급받은 Access Token을 입력하세요.
GitHub Pages URL: https://kunmade88.github.io/kpi-planner/
STEP 2 — GOOGLE SHEETS (Apps Script)
2
Apps Script Web App URL 입력
Google Sheets에서 Apps Script → 배포 → 웹 앱으로 배포 후 URL을 입력하세요.
✅ 이 방법은 OAuth 없이 Sheets와 양방향 연동 가능합니다!
💡 Apps Script 배포 방법:
Sheets → 확장 프로그램 → Apps Script → 아래 코드 붙여넣기 → 배포 → 새 배포 → 웹 앱 → 액세스: 모든 사용자 → 배포
